1. Introduction

This is a regular paragraph. Content flows with bold emphasis and inline code for technical terms. The font and spacing match the ExamForge app.

Definition

Pharmacokinetics — The study of how the body absorbs, distributes, metabolizes, and excretes drugs.

Key Point

This is an important concept that students should remember for the exam.

Formula
C = C₀ × e−kt
Where C = concentration at time t, C₀ = initial concentration, k = elimination rate constant
Important Note

This distinction is commonly tested in competitive exams. Pay special attention to the difference.

Example

Problem: Calculate the half-life of a drug with k = 0.1 hr⁻¹.

Solution: t½ = 0.693 / k = 0.693 / 0.1 = 6.93 hours

Practice Questions

MCQ Easy
Q1. Which of the following best describes first-order kinetics?
  • A. Constant amount eliminated per unit time
  • B. Constant fraction eliminated per unit time
  • C. Rate independent of concentration
  • D. Enzymes are saturated
Correct: B. In first-order kinetics, a constant fraction (not amount) of the drug is eliminated per unit time. The rate of elimination is proportional to the drug concentration.
